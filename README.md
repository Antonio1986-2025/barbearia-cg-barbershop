# Barbearia Campo Grande Barbershop

Site institucional da barbearia tradicional **Campo Grande Barbershop** — desde 2012 no coração de Campo Grande, MS.

## Estrutura

```
├── index.html    (481 linhas — estrutura e conteúdo)
├── style.css     (1.062 linhas — estilo visual)
├── script.js     (271 linhas — interatividade)
├── README.md
└── package.json
```

## Funcionalidades

- Cardápio de serviços (6 categorias)
- Galeria de imagens com fotos reais da barbearia
- Equipe com fotos dos barbeiros (Roger, Eduardo, Victor)
- Agendamento em 4 etapas (serviço → barbeiro → data/horário → WhatsApp)
- Depoimentos reais do Trinks (avaliação 5.0 ★)
- Mapa interativo com localização
- Modal de agendamento com validação
- Botão flutuante do WhatsApp
- Design responsivo (mobile-first)
- Meta tags Open Graph e JSON-LD (schema.org)

## Como usar

```bash
npm run serve     # Servidor local com npx serve
```

Acesse `http://localhost:3000`

## Deploy

Hospedado no **GitHub Pages** — push na branch `master` dispara deploy automático.
